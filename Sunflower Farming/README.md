# Sunflower Farming

## Prerequisites

- 32x32 World Unlocked
- 32 Drones Unlocked

## Implementation

Very simple codes, it spins up a drone for each column in the world, which tills the ground, waters and harvests. This ignores the sunflowers special mechanic, having a different number of petals, as the scale of the farm is more than enough to unlock the "Big Power Farmer" achievement. You could make a much higher output farm than this.

It may take a few loops to unlock, as the drones need to build up speed using the harvested power.
